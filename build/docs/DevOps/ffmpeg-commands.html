    FFMpeg

    # 使用N卡加速 mp4 -> hls

    ffmpeg -i input.mp4 -c:v h264_nvenc -preset medium -b:v 2000k -maxrate:v 2000k -bufsize:v 4000k -s:v 1280x720 \n
-map 0:v:0   -map 0:a? -c:a copy -f ssegment \n
-segment_list ./hls/out_720p.m3u8 -segment_time 10 ./hls/out_%03d_720p.ts
    1
    2
    3
    这个ffmpeg命令用于将输入的input.mp4视频文件转码为使用NVIDIA GPU加速的H.264编码，并且将其分割成多个HLS（HTTP Live Streaming）的.ts片段，同时生成一个播放列表out.m3u8。下面是该命令各部分的详细解读：

基本输入输出：
-i input.mp4：指定输入文件为input.mp4。
视频编码：
-c:v h264_nvenc：使用NVIDIA的GPU加速的H.264编码器h264_nvenc。
-preset medium：设置编码的预设为medium，这通常是一个速度和质量的折衷。
-b:v 2000k：设置视频比特率为2000kbps。
-maxrate:v 2000k：设置视频最大比特率为2000kbps，防止视频比特率超过这个值。
-bufsize:v 4000k：设置VBV（Video Buffering Verifier）缓冲区大小为4000kbps，这有助于控制视频流的稳定性。
-s:v 1280x720：设置视频分辨率为1280x720，即720p。
流映射：
-map 0:v:0：从输入文件（编号为0）选择第一路视频流（编号为0）进行转码。
-map 0:a?：如果输入文件包含音频流，则选择所有的音频流（如果有多个音频流，它们都会被包含）。?表示如果音频流不存在，则不会报错。
音频编码：
-c:a copy：对于音频流，不进行重新编码，而是直接复制，以保持原始音频的质量和格式。
输出格式和分段：
-f ssegment：指定输出格式为ssegment，即分割的媒体流格式，用于HLS。
-segment_list ./hls/out.m3u8：生成一个播放列表文件out.m3u8，用于HLS。这个文件会被保存在./hls/目录下。
-segment_time 10：每个分段文件的时长为10秒。
./hls/out_%03d_720p.ts：输出的.ts文件模板，它们会被保存在./hls/目录下。%03d是一个占位符，表示分段的序号，从000开始。文件名中的720p正确地反映了视频的实际分辨率。
这个命令通常用于视频流服务，如在线视频直播或点播，其中视频内容需要被分割成多个小的片段，以支持流媒体的播放。通过HLS，这些片段可以被有效地传输到用户的设备上，并支持在多种网络条件下的播放。

    # N卡加速 mp4 -> flv

    ffmpeg -hwaccel auto -i input.mp4 -c:v h264_nvenc -preset fast -c:a copy output.flv
    1
    解释一下这个命令：
-hwaccel auto：自动选择合适的硬件加速方法，这里会使用NVIDIA GPU如果可用。
-i input.mp4：指定输入的MP4文件。
-c:v h264_nvenc：使用NVIDIA的H.264 NVENC编码器进行视频编码。
-preset fast：选择一个较快的编码预设，平衡速度和输出质量。根据需要，你也可以选择其他预设如medium, slow等。
-c:a copy：音频流直接复制，不重新编码，这样可以节省时间和保持原始音质。
output.flv：指定输出的FLV文件名。

    ffmpeg -hwaccel auto -i input.mp4 -c:v h264_nvenc -preset fast -b:v 500k -s 640x360 -c:a copy output_360p.flv
    1
    解释一下各参数的含义：
-hwaccel auto：自动选择硬件加速方法，这里会根据系统配置自动使用NVIDIA GPU（如果可用）。
-i input.mp4：指定输入文件为input.mp4。
-c:v h264_nvenc：指定使用NVIDIA NVENC硬件加速编码器进行H.264视频编码。
-preset fast：使用快速编码预设，平衡编码速度和输出质量。根据需要，你也可以选择其他预设如medium, slow, hp, hq等。
-b:v 500k：设置视频比特率为500kbps。根据你的需求调整这个值来控制输出视频的质量和文件大小。
-s 640x360：设置输出视频的分辨率为640x360。你可以根据需要替换为其他分辨率，比如1280x720。
-c:a copy：音频流直接复制，不进行重新编码，这样可以保持原音质且加快处理速度。
output.flv：指定输出文件名为output.flv，即FLV格式。
